Welcome to the Introduction to Python Programming Course. This directory, PythonClass, is where you find support files related to the class exercises. It is also where you will be doing your work.

  • The Exercises/ folder is where you find all the class exercises.

  • The Data/ folder is where you find data files, scripts, and other files used by the exercises.

  • The Ext/ folder contains files related to Python extension programming.

  • The Solutions folder contains complete solution code for various exercises. Please see the README file in that directory for more details.

We will be using a program known as IDLE to edit and run Python programs. In this folder, you will find a script called RunIDLE. On Windows, you can just double-click on this file and IDLE will start. On Macintosh, you will want to run this from a terminal by going into the PythonClass/ folder and then typing python RunIDLE.pyw.

If you need to download Python for some reason, it can be obtained at www.python.org.

Some Rules of the Road

  1. Every program that you create MUST be placed in the top level PythonClass folder (the same folder where this README file was found). NO EXCEPTIONS

  2. You will NOT create any file or program in any folder or directory other than the top level PythonClass folder (see rule 1).

  3. If you are still inclined to create programs in any folder other than the PythonClass folder, you will refer to Rule 1 and Rule 2 above.

  4. You will ask for help at the first sign of any problem with your machine, IDLE, or Python.

  5. Some exercises are more difficult than others. However, solutions are provided for ALL class exercises. There is no excuse for not having working code--cut and paste from the solution if you have to.

  6. You will EXACTLY follow the filename conventions listed in each exercise. That is, if the exercise asks you to create a program called spam.py, you will create a program called spam.py, not spam2.py,spammy.py, or processedmeat.py.

  7. If you have never used Python before, you will edit and run all programs using IDLE. You will not use any other editor such as vi, emacs, vim, pico, Eclipse, Textmate, or Windows Notepad.
